Atcoder [ABC166D] I hate Factorization

题目传送门

题意如下

找一组,满足;

分析

如果直接暴力枚举，相信我，一定会TLE(试试就逝世)
通过复杂的手算，我们可以发现：
  1. 1.的差越小，也越小
  2. 2.由于已经大于，所以的范围就是

代码

#include<bits/stdc++.h>
#define ll long long
using namespace std;
ll y;
ll f(ll x){return x*x*x*x*x;};//5次方函数
int main(){
    cin>>y;
    //枚举
    for(int i=-150;i<=150;i++){
        for(int j=-150;j<=150;j++){
            if(f(i)-f(j)==y){//如果i^5-j^5==Y
                printf("%d %d",i,j);
                return 0;
            }
        }
    }
    return 0;
}
